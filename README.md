## Installation
- `git clone https://github.com/francijpg/rn-layout-native`
- Before running the application (step 5), first you must install some dependencies:
  1. `cd rn-layout-native`
  2. `yarn install`
  3. `cd ios`
  4. `pod install`
  5. `npm run ios`
  6. `npm run android`
